# 📘 แนวทางการอัปโหลดโจทย์จากภาพ (สำหรับระบบคลังข้อสอบ)

เอกสารนี้ใช้เพื่อเป็นแนวทางในการส่งภาพโจทย์ให้ระบบแปลงเป็น JSON โดยอัตโนมัติ พร้อมใช้งานในระบบคลังข้อสอบออนไลน์

---

## ✅ ขั้นตอนเมื่ออัปโหลดภาพโจทย์

### 1. ตรวจสอบเนื้อหาในภาพ
- ✔️ เป็นภาพเฉพาะส่วนของโจทย์ (ไม่ควรมีคำเฉลย ลายมือ สัญลักษณ์ตรวจ ฯลฯ)
- ✔️ เป็นภาพของโจทย์ทีละข้อ (หากมีหลายข้อ ให้แยกภาพ)
- ✔️ หากมีภาพประกอบ ควรอยู่ในรูปเดียวกันกับโจทย์

### 2. ระบบจะทำการ:
- อ่านเนื้อหาโจทย์และแปลงเป็น JSON
- แปลเป็นภาษาอังกฤษ
- แปลงเศษส่วน/สมการให้เป็น HTML (ถ้ามี)
- สร้างรหัส `code`, `id`, และ `attributes` ให้พร้อม
- ใส่คะแนน (`score`) โดยอัตโนมัติ

### 3. คุณสามารถตรวจสอบและอนุมัติ:
- ถ้าถูกต้อง → ระบบจะขยายโจทย์เพิ่มอีก 10 ข้อ (variant)
- รวมทั้งหมด 11 ข้อในชุดเดียว พร้อม export JSON ทันที

---

## 🧠 รูปแบบโครงสร้าง JSON ที่ใช้

```json
{
  "id": "UUID",
  "code": "p6-chula-2022-o-qXXX",
  "textTh": "...",
  "textEn": "...",
  "textThHtml": "...",
  "textEnHtml": "...",
  "image": null,
  "choices": [],
  "answer": null,
  "shortAnswer": ["..."],
  "attributes": {
    "year": "2022",
    "topic": ["topic:..."],
    "skill": ["skill:..."],
    "trap": ["trap:..."],
    "difficulty": 1,
    "questionNo": "XXX",
    "source": "source:chulabhorn",
    "examSet": "chula-p6-2022"
  },
  "score": 1,
  "source": "Chulabhorn Science Math Test 2022",
  "derivedFrom": null,
  "estimatedTimeSec": 60,
  "ownerOrg": "me",
  "createdBy": "chatgpt",
  "updatedBy": "chatgpt"
}
```

---

## 📌 หมายเหตุ:
- `questionNo` ใส่เฉพาะในข้อสอบจริงเท่านั้น
- `score` จำเป็นต้องมีเสมอ (default = 1)
- `derivedFrom` ใช้ในข้อ variant เท่านั้น
- `image` จะเป็น path เช่น `/images/questions/p6-chula-2022-o-q003.png` ถ้ามีภาพ
