⚡ เหตุการณ์พิเศษ

ระหว่างเกมจะมีโจทย์พิเศษโผล่ขึ้นมา — แย่งกันตอบให้เร็วที่สุด!
ถ้าตอบถูกจะได้รับ Sabotage หรือ Chaos effect ทันที

📡 Online Mode

1. ผู้เล่นคนแรก สร้างห้อง → รับ Room Code
2. แชร์โค้ดให้เพื่อน → เพื่อนกด เข้าร่วม
3. Host กด เริ่มเกม เมื่อครบผู้เล่น
⚠️ ทุกคนต้องเข้าจาก URL เดียวกัน (ไม่ใช่ localhost ต่างเครื่อง)
